简介

Grenade Helper 是一款使用 Flutter 开发的 CS2 投掷物辅助工具。它可以帮助玩家快速学习和掌握各种投掷物的投掷点位，提高游戏水平。

本工具支持Android端和iOS端，以及Windows桌面端

本工具完全开源免费，基于 GPL-3.0 协议发布。

安装指南

Windows

  1. 从主页下载最新版本的安装包
  2. 如果是 .zip 文件，解压到任意目录
  3. 运行 grenade_helper.exe 即可启动

Android

  1. 从主页下载最新版本的安装包
  2. 如果是 .zip 文件，解压到任意目录
  3. 将 grenade_helper.apk 安装到手机上

iOS

  1. 从主页下载最新版本的安装包
  2. 如果是 .zip 文件，解压到任意目录
  3. 侧载 grenade_helper.ipa 即可启动
  4. 如果遇到签名问题，请参考 iOS 侧载方法
💡 提示：Windows用户如果遇到 Windows Defender 拦截，请点击"更多信息"并选择"仍要运行"。这是因为软件没有数字签名导致的。

iOS 侧载方法

由于 iOS 系统限制，安装第三方应用需要通过侧载方式。以下是几种常用的侧载方法：

方法一：使用 AltStore（推荐）

AltStore 是目前最流行的 iOS 侧载工具，支持 Windows 和 macOS，操作简单直观。

所需软件

  • AltServer：https://altstore.io/
  • iTunes（Windows 用户需要）：Apple 官网下载（必须是官网版本，不能用 Microsoft Store 版本）
  • iCloud（Windows 用户需要）：Apple 官网下载

安装步骤

  1. 在电脑上下载并安装 AltServer
  2. iOS 16+ 用户：前往 设置 → 隐私与安全性 → 开发者模式，开启并重启设备
  3. Windows 用户需要同时安装 iTunes 和 iCloud（必须从 Apple 官网下载，不能用 Microsoft Store 版本）
  4. 运行 AltServer，它会在系统托盘中显示
  5. 将 iPhone 通过数据线连接到电脑
  6. 在手机上选择"信任此电脑"
  7. 点击系统托盘中的 AltServer 图标 → Install AltStore → 选择你的设备
  8. 输入你的 Apple ID 和密码（用于签名应用）
  9. 安装完成后，在 iPhone 上打开 设置 → 通用 → VPN与设备管理，信任开发者证书
  10. 安装 AltStore 完毕之后，点击 Settings 页面，选择顶部的 Sign in with Apple ID，登录你的 Apple ID
  11. 登录完毕之后在 AltStore 中点击左上角 + 号，选择下载好的 .ipa 文件进行安装

注意事项

  • 一个个人的 Apple ID 无法签名超过 10 个应用，如果超过 10 个应用，需要重新签名，且一台设备只能同时安装包括AltStore在内的最多3个应用，以下包括所有的侧载方法都是如此
  • AltStore 相关问题请查阅 AltStore 官网的说明文档
⚠️ 注意：使用 AltStore 安装的应用有效期为 7 天，到期后需要重新签名。请保持手机和电脑在同一网络环境下，并保持 AltServer 在后台运行，可以自动续签。

方法二：使用 SideStore

SideStore 是 AltStore 的一个分支，最大的特点是无需电脑即可续签（只需首次安装时使用电脑）。需要额外安装 LocalDevVPN。

前置要求

  • iOS 14.0 或更高版本（支持 iOS 18）
  • 一台 Windows 或 macOS 电脑（仅首次安装需要）
  • Apple ID（建议使用小号，避免主号被风控）
  • iOS 16+ 需要开启开发者模式

所需软件

  • SideServer：https://sidestore.io/（电脑端安装）
  • LocalDevVPN：在 App Store 搜索下载（iPhone 端安装）
  • iTunes（Windows 用户需要）：Apple 官网下载

安装步骤

第一部分：电脑端操作

  1. 从官网下载 SideServer 并安装到电脑
  2. Windows 用户：确保已安装 iTunes（官网版本），否则 SideServer 无法识别设备
  3. 使用数据线将 iPhone 连接到电脑
  4. iPhone 上弹出"信任此电脑"提示时，点击信任并输入锁屏密码
  5. 运行 SideServer，等待识别到你的设备
  6. SideServer 会自动生成配对文件（Pairing File），这是续签的关键文件
  7. 在 SideServer 中点击 "Install SideStore" 按钮
  8. 输入你的 Apple ID 和密码（用于签名，建议使用小号）
  9. 等待安装完成，SideStore 应用会出现在 iPhone 主屏幕上

第二部分：iPhone 端操作

  1. iOS 16+ 用户：前往 设置 → 隐私与安全性 → 开发者模式，开启并重启设备
  2. 打开 iPhone 设置 → 通用 → VPN与设备管理
  3. 找到你的 Apple ID 对应的开发者描述文件，点击信任
  4. 打开 App Store，搜索并下载 LocalDevVPN
  5. 打开 LocalDevVPN 应用，点击连接开启 VPN
  6. 打开 SideStore，如提示添加 VPN 配置请点击允许
  7. 使用同一个 Apple ID 登录 SideStore
  8. 点击右上角 + 号，选择下载好的 .ipa 文件进行安装

续签方法

SideStore 的最大优势是可以在手机上直接续签，无需连接电脑：

  1. 打开 LocalDevVPN 应用，开启 VPN
  2. 打开 SideStore 应用
  3. 在"我的应用"页面，点击需要续签的应用旁边的刷新按钮
  4. 或者点击右上角菜单，选择"刷新全部"
💡 提示：
  • 建议在应用到期前 1-2 天进行续签，避免应用过期无法打开
  • 续签时 无需连接电脑或同一 WiFi
  • 普通 Apple ID 签名的应用有效期为 7 天
⚠️ 常见问题：
  • SideServer 识别不到设备：确保安装了官网版 iTunes，重新插拔数据线，重启 SideServer
  • 安装时报错 -22421：Apple ID 被风控，等待几天或更换 Apple ID
  • 应用闪退：证书已过期，需要重新续签
  • 续签失败显示 "No WiFi or VPN"：确保 LocalDevVPN 已开启

方法三：使用 Sideloadly

Sideloadly 是一个简单易用的 IPA 安装工具，无需安装 AltStore。

所需软件

  • Sideloadly：https://sideloadly.io/
  • iTunes（Windows 用户需要）

安装步骤

  1. 下载并安装 Sideloadly
  2. 将 iPhone 通过数据线连接到电脑
  3. 打开 Sideloadly，选择你的设备
  4. 将 .ipa 文件拖入 Sideloadly 窗口
  5. 输入你的 Apple ID 和密码
  6. 点击 Start 开始安装，点击 Start 旁边的刷新按钮，即可开启自动续签
  7. 安装完成后，在 iPhone 上信任开发者证书
⚠️ 注意：使用 Sideloadly 安装的应用有效期为 7 天，到期后需要重新签名。请保持手机和电脑在同一网络环境下，并保持 Sideloadly 在后台运行并定期连接手机，可以自动续签。

方法四：使用爱思助手/3uTools

国内用户可以使用爱思助手或 3uTools，操作更简单。

所需软件

  • 爱思助手：https://www.i4.cn/
  • 3uTools：https://www.3u.com/

安装步骤（以爱思助手为例）

  1. 下载并安装 爱思助手
  2. 将 iPhone 连接到电脑
  3. 打开爱思助手，等待识别设备
  4. 点击 应用游戏 → 导入安装
  5. 选择 .ipa 文件进行安装
  6. 某些情况下可能需要信任证书
💡 提示：如果你有付费的开发者账号（$99/年），签名的应用有效期为 1 年。普通 Apple ID 签名的应用有效期为 7 天。

使用方法

基本操作

  1. 启动软件后，选择你想要查看的地图
  2. 在地图上启动编辑模式后，点击添加你想要的投掷物点位
  3. 点击点位即可查看详细的投掷教程，包括站位、准星位置和投掷方式
  4. 在游戏中使用时，直接点击点位查看即可

悬浮窗功能

软件在Windows端支持悬浮窗模式，使用快捷键Alt+G即可调出，悬浮窗可以在游戏时保持窗口置顶，方便随时查看投掷物教程。

功能模块详解

以下是 Grenade Helper 各功能模块的详细说明：

🗺️ 地图模块

地图模块是应用的核心功能之一，提供全面的地图浏览和管理能力。

  • 地图选择：主页提供所有 CS2 竞技地图的快速入口
  • 地图缩放：支持鼠标滚轮（桌面端）或手势（移动端）缩放地图，查看不同比例
  • 点位标记：地图上以图标形式展示所有投掷物点位
  • 阵营筛选：可按 CT/T 阵营筛选显示的点位
  • 收藏功能：支持添加常用道具，直接在地图界面底栏显示，同时在地图上高亮显示，方便用户快捷查看
  • 出生点显示：支持显示每张地图拥有的出生点，方便用户使用

🎯 投掷物点位模块

点位模块用于管理和查看具体的投掷物教程。

  • 点位详情：点击地图上的点位查看详细投掷教程
  • 多媒体支持：支持图片和视频形式的教程内容
  • 投掷方式：标注左键/右键/左右键等投掷方式
  • 点位名称：支持自定义点位名称
  • 点位类型：支持自定义点位类型，可以选择分类为烟闪雷火以及CT/T阵营独有道具
  • 点位详细步骤：支持自定义详细步骤和步骤名称，以及站位、准星位置等
  • 点位快捷移动：支持长按拖拽移动或者点位页面使用单独的移动功能，直接点选地图快速移动。
  • 编辑功能：支持添加、编辑、左滑快捷删除自定义点位

🔍 搜索模块

搜索模块帮助用户快速定位需要的投掷物点位。

  • 全局搜索：主页支持跨地图搜索所有点位
  • 地图内搜索：在地图界面搜索当前地图的点位
  • 关键词匹配：支持按点位名称、描述等进行搜索

🔄 分享/导入模块

分享导入功能方便用户之间交流和多设备同步。

  • 导出点位：支持将选中的点位、每张地图的道具以及全部地图的点位导出为可分享的文件
  • 导入点位：从文件导入他人分享的点位数据
  • 批量操作：支持批量导入导出，无需逐个操作

🖼️ 悬浮窗模块（Windows 专属）

悬浮窗功能让你可以在游戏中实时查看投掷物教程。

  • 快捷键呼出：使用 Alt+G 快速呼出/隐藏悬浮窗
  • 窗口置顶：悬浮窗始终保持在游戏窗口之上
  • 实时同步：与主应用同步显示当前查看的点位
  • 透明度调节：可调节悬浮窗透明度以适应游戏画面

常见问题

Q: 软件是否免费？

A: 是的，Grenade Helper 完全免费，并且会不断更新。

Q: 软件会不会被 VAC 封禁？

A: 不会。本软件只是一个教学工具，不会读取或修改游戏内存，不会影响游戏文件，完全安全。

Q: 如何添加自定义点位？

A: 软件支持添加自定义点位。在地图界面点击添加按钮，按照提示输入点位信息即可。

Q: 支持哪些平台？

A: 目前主要支持 Windows 桌面平台。其他平台（如 macOS、Linux）正在开发中。
